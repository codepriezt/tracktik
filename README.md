## Installation

### Clone repo

``` bash
# clone the repo
$ git clone https://github.com/codepriezt/tracktik.git tracktik

# go into app's directory
$ cd tracktik

# install app's dependencies
$ npm install
```

### Basic usage

Run `build` to build the project. The build artifacts will be stored in the `build/` directory.
